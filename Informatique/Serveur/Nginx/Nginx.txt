====== Nginx ======
Created mardi 16 mai 2017

* autoriser les requêtes Cross Origin sur les polices de caractères : ajouter cette configuration dans la partie ''server'' :
{{{code: lang="def" linenumbers="True"

}}}
location ~* \.(eot|ttf|woff|woff2)$ {
	add_header Access-Control-Allow-Origin *;
}
source: https://serverfault.com/questions/162429/how-do-i-add-access-control-allow-origin-in-nginx*
