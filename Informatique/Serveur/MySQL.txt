===== Old passwords =====

* [[MySQL]] a changé la façon de stocker les mots de passe, et PHP a suivi. Il arrive parfois que PHP essaie de se connecter en utilisant nouvelle méthode, alors que [[MySQL]] stocke encore ses mots de passe selon l'ancienne. Pour mettre à jour le mot de passe d'un utilisateur, il est nécessaire de le redéfinir manuellement. Voici un exemple de procédure rapide :
	* Vérifier la a liste des utilisateurs concernés:
		SELECT User, Host, Password FROM mysql.user WHERE (plugin = '' AND LENGTH(Password) = 16) OR plugin = 'mysql_old_password';
	* Mettre à jour un par un les utilisateurs concernés
		UPDATE mysql.user SET plugin = 'mysql_native_password',Password = PASSWORD('new_password') WHERE (User, Host) = ('user_name','localhost');
	* Rafraîchir les privilèges :
		FLUSH PRIVILEGES;
